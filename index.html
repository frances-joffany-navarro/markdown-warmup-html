Hypertext Markup Language (HTML)

is the standard markup language for documents designed to be displayed in a web browser. It can be assisted by technologies such as Cascading Style Sheets (CSS) and scripting languages such as JavaScript.

HTML Tags

Unordered list

To create an unordered list use <ul> tag.

HTML Result

<ul>
<li> Item 1 </li>
<li> Item 2 </li>
<li> Item 3 </li>
</ul>

  • Item 1
  • Item 2
  • Item 3

Ordered list

To create an ordered list use <ol> tag.

HTML Result

<ol>
<li> Last Name </li>
<li> First Name </li>
<li> Age </li>
</ol>

  1. Last Name
  2. First Name
  3. Age

List within a list

HTML Result

<ol>
<li> Profile </li>
<ul>
<li> Last Name </li>
<li> First Name </li>
<li> Age </li>
</ul>
<li> Document Needed </li>

Titles and Subtitles

Headers are used to emphasize any titles and subtitles of a page

<h1> tag is use for the title of the page

<h2> tag is use for subtitles
