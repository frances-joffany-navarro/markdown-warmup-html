Hypertext Markup Language (HTML)

is the standard markup language for documents designed to be displayed in a web browser. It can be assisted by technologies such as Cascading Style Sheets (CSS) and scripting languages such as JavaScript.

Example of HTML Codes

Unordered list

To create an unordered list use <ul> tag.

Code:

<ul>
<li> Item 1 </li>
<li> Item 2 </li>
<li> Item 3 </li>
</ul>

Result:

  • Item 1
  • Item 2
  • Item 3

Ordered list

To create an ordered list use <ol> tag.

Code:

<ol>
<li> Last Name </li>
<li> First Name </li>
<li> Age </li>
</ol>

Result:

  1. Last Name
  2. First Name
  3. Age

List within a list

Code:

<ol>
<li> Profile </li>
<ul>
<li> Last Name </li>
<li> First Name </li>
<li> Age </li>
</ul>
<li> Document Needed </li>
<ol>
<li> Identity Card </li>
<li> Bank Card </li>
<li> Registration Form </li>
</ol>
</ol>

Result:

  1. Profile
    • Last Name
    • First Name
    • Birthday
  2. Document Needed
    1. Identity Card
    2. Bank Card
    3. Registration Form

Links

In creating links in HTML you use the <a> tag.

Syntax:
<a href= "https://github.com/frances-joffany-navarro" > My github account </a> <a href= "https://linkedin.com/in/frances-navarro-05775a126" > My Linkedin Bio </a> <p>Learn more about the organization <a href="becode.org">Becode</a></p>

Examples:

My github account
My Linkedin Bio

Learn more about the organization Becode

Static image

Syntax:
<img src= "/image.jpg" alt= "image">
